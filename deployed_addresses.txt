# Deployed contracts on Rinkeby:

- IuvoCoreV1: [0x675f8bb31c7689df60e7c019e0c4fe9d0d8052f2](https://rinkeby.etherscan.io/address/0x675f8bb31c7689df60e7c019e0c4fe9d0d8052f2)
- PausableProxy: [0x5f17193cda988ddecb1c2255c1c4e8077ecc8b66](https://rinkeby.etherscan.io/address/0x5f17193cda988ddecb1c2255c1c4e8077ecc8b66)

All interaction with the core contract must happen through the proxy. See below for na example with web3js.

```
let IuvoCore = web3.eth.contract(<abi array here>)
let iuvoCoreByProxy = IuvoCore.at('0x5f17193cda988ddecb1c2255c1c4e8077ecc8b66') // Note that this is the proxy address, not the iuvocore address.

iuvoCoreByProxy.someMethod(someCallback)
```
